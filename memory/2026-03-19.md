  ```python

      def add_task(example):
          example["task"] = "Pick up the object"
          return example
      hf_dataset = hf_dataset.map(add_task)
  ```

## 📝 Lessons Learned

1. **Disk Space Planning**: 30GB insufficient for training with checkpoints
   - Recommendation: Use 50GB+ disk
   - Or use `--save_freq 10000` to save only at end

2. **Data Format Compatibility**: Always test data access pattern before running full evaluation
   - LeRobotDataset returns tuple, not dict
   - Must access `dataset[idx][0]` for data dict

3. **Skill Activation**: Always check for existing skills before manual execution
   - Saves time and ensures correct workflow
   - Follows user's established preferences

4. **Progress Reporting**: Critical for user experience
   - Silent execution = user thinks bot is stuck
   - Regular updates build trust and transparency
