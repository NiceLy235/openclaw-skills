**补充提交昨日日志：**
```bash
git add memory/2026-03-19.md

git push origin master
```

**设置自动同步：**
```bash
openclaw cron add \
  --name "每日记忆同步" \
  --cron "0 23 * * *" \
  --session isolated \
  --agent main \
  --model glm-5 \
  --message "请同步今天的记忆到 GitHub：1) 检查 memory/ 和 MEMORY.md 是否有更新；2) 如果有更新，提交并推送到 GitHub；3) 报告同步结果" \
  --tz "Asia/Shanghai" \
  --no-deliver
```
