## 💡 Key Lessons Learned

### 1. V2Ray Version Matters
- Always check V2Ray version before using
- apt install gives outdated 4.34.0 with bugs
- Must use official install script for 5.44.1+

### 2. PyTorch Installation
- pip is more reliable than conda for PyTorch GPU
- Use background nohup scripts for long installations
- Always verify CUDA availability after installation

### 3. Dataset Merging
- LeRobot's `lerobot_edit_dataset` tool requires specific parameters
- Custom Python script more flexible for complex merging
- Always verify merged dataset structure (meta/info.json)

### 4. Dependency Management
- Always check dependency conflicts before training
- LeRobot 0.4.1 has strict huggingface-hub requirements
- May need separate environments for different tasks
